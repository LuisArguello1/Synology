rsync es un protocolo de transferencia de datos que puede utilizarse para realizar copias de seguridad remotas de forma incremental y eficiente.

rsync utilizará este puerto para transferencias cifradas

Las cuentas rsync permiten que usuarios externos (sin acceso DSM) puedan realizar copias de seguridad en el NAS usando el protocolo rsync.

Límites de velocidad:

Configure límites de velocidad para el tráfico de rsync. Esto ayuda a evitar la saturación del ancho de banda durante las transferencias.

Observación

  • Asegúrese de que el puerto SSH esté abierto en el firewall
  • Para copias de seguridad, configure las tareas de rsync en el panel de respaldo
  • Los límites de velocidad ayudan a mantener el rendimiento del NAS durante transferencias grandes
